# EdgeTapeOverlay
화면 테두리 오버레이 프로그램

## ⚠️ 중요 안내사항

### 프로그램 목적
본 프로그램은 **3D 멀미를 경험하는 사용자를 위한 임시 보조 도구**입니다.
- 화면 가장자리에 시각적 기준점을 제공하여 멀미 완화를 시도합니다
- 모든 사용자에게 효과가 있는 것은 아니며, 개인차가 있을 수 있습니다

### 사용 제한 및 면책
- ⚠️ **게임 등 본래 용도에 맞지 않는 목적으로 사용해서는 안 됩니다**
- ⚠️ **본 프로그램 사용으로 인한 게임 이용 제재(밴, 정지 등)에 대한 책임은 전적으로 사용자 본인에게 있습니다**
- ⚠️ **안티치트 프로그램과의 충돌, 게임 정책 위반 등의 가능성을 충분히 인지하고 사용하시기 바랍니다**
- 개발자는 본 프로그램 사용으로 인한 어떠한 문제에 대해서도 책임지지 않습니다

### 전체 화면 프로그램에서는 정상적으로 작동하지 않을 수 있습니다.
-  [전체 창모드] 또는 [경계 없는 전체화면] 으로 설정해주세요.**

## 시스템 요구사항
- Windows 10 이상
- .NET Framework 4.7.2 이상

## 구성 파일
- `EdgeTapeOverlay.exe` - 실행파일
- `Config.json` - 설정파일 (자동 생성)

## 사용법

### 1. 프로그램 실행
`EdgeTapeOverlay.exe` 실행 시 시스템 트레이에 아이콘이 표시됩니다.

### 2. 기본 조작
**시스템 트레이 아이콘**
- 더블클릭: 모든 오버레이 켜기/끄기
- 우클릭: 메뉴 표시

**트레이 메뉴**
- 🔲 테두리 켜기/끄기
- ➕ 크로스헤어 켜기/끄기
- 🔳 모두 켜기/끄기
- ⚙️ 설정 변경
- ❌ 종료

### 3. 설정 기능

#### 표시 모니터
- 단일 모니터: 주 모니터만 표시
- 듀얼 모니터: 주 모니터 / 보조 모니터 / 모든 모니터 선택 가능
- 3개 이상: 각 모니터 개별 선택 가능

#### 테두리 모드
- 🔲 사각형 전체 테두리: 화면 가장자리 전체에 테두리 표시
- ➕ 십자(+) 형태 테두리: 화면 상하좌우 중앙에 십자 형태로 표시

#### 테두리 외관 설정
- 테두리 색상: 원하는 색상으로 변경 가능
- 테두리 두께: 1~50px 조절 가능
- 십자 길이: 50~500px 조절 가능 (십자 모드 시)
- 테두리 투명도: 0~100% 조절 가능

#### 크로스헤어 설정
- 표시 스타일
  - ⭕ 표시 안함
  - ➕ 십자선
  - ⚫ 점
  - ⬜ 사각 박스
- 크로스헤어 색상: 원하는 색상으로 변경 가능
- 크로스헤어 투명도: 0~100% 조절 가능

#### 색상 변경
- 테두리 색상 변경: 테두리만 색상 변경
- 크로스헤어 색상 변경: 크로스헤어만 색상 변경
- 🎨 테두리 + 크로스헤어 색상 통일: 모든 요소를 동일한 색상으로 변경
- 사용자 지정 색상 16개 저장 가능

### 4. 설정 저장
- 모든 설정은 자동으로 `Config.json` 파일에 저장됩니다
- 프로그램 재시작 시 이전 설정이 자동으로 적용됩니다
- 설정 파일은 실행 파일과 동일한 폴더에 생성됩니다

## 주요 특징

### 완벽한 투명도 지원
- Per-Pixel Alpha 렌더링 방식 사용
- 투명도 설정에 따라 화면과 자연스럽게 블렌딩
- 배경 간섭 없이 깔끔한 표시

### 독립적인 제어
- 테두리와 크로스헤어를 개별적으로 켜기/끄기 가능
- 각각 다른 색상과 투명도 설정 가능

### 멀티 모니터 지원
- 최대 10개 모니터까지 지원
- 모니터별 개별 선택 가능
- 동적으로 연결된 모니터 감지

### 포터블 프로그램
- 설치 불필요
- 실행 파일과 설정 파일만으로 구성
- USB 등으로 이동하여 다른 컴퓨터에서도 동일한 설정으로 사용 가능

## 문제 해결

### 오버레이가 표시되지 않는 경우
1. 설정에서 올바른 모니터가 선택되었는지 확인
2. 투명도가 0%로 설정되어 있지 않은지 확인
3. 다른 항상 위 프로그램과의 충돌 가능성 확인

### 설정이 저장되지 않는 경우
1. `Config.json` 파일에 대한 쓰기 권한 확인
2. 실행 파일이 있는 폴더가 읽기 전용이 아닌지 확인

### 안티치트 프로그램 경고
- 일부 게임의 안티치트가 오버레이를 감지할 수 있습니다
- **게임 실행 전 프로그램을 종료하는 것을 권장합니다**
- 문제 발생 시 즉시 사용을 중단하시기 바랍니다

## 기술 정보

### 렌더링 방식
- WinForms Layered Window
- UpdateLayeredWindow API를 통한 Per-Pixel Alpha
- 마우스 클릭 통과 (WS_EX_TRANSPARENT)
- 항상 최상위 표시 (WS_EX_TOPMOST)

### 시스템 영향
- CPU/GPU 사용량: 최소
- 메모리 사용량: 약 10~20MB
- 화면 갱신: 설정 변경 시에만 발생
